## Initialize the Client

The Tonight Pass SDK provides a unified client for accessing all API endpoints. Initialize the client with your configuration:

<Steps>
  <Step title="Import the TonightPass class">
    
    Import the main `TonightPass` class from the `tonightpass` package:

    ```typescript
    import { TonightPass } from 'tonightpass';
    ```

  </Step>

  <Step title="Create a new instance">
    
    Create a new instance of the client with your configuration. The client accepts a `ClientOptions` object:

    ```typescript
    const tnp = new TonightPass({
      baseURL: 'https://api.tonightpass.com',
      apiKey: 'your-api-key', // Optional
      accessToken: 'your-access-token', // Optional
      cache: { // Optional caching configuration
        enabled: true,
        ttl: 60000 // Cache TTL in milliseconds
      }
    });
    ```

    <Note>
    Currently, API keys are for internal use only. Future versions will support public API keys.
    </Note>

  </Step>

  <Step title="Make your first API call">
    
    Let's start with a simple health check to verify the API is accessible:

    ```typescript
    // Check API health
    const health = await tnp.health.api();
    console.log('API Status:', health);
    ```

    You can also check the database and app health:

    ```typescript
    // Check database health
    const dbHealth = await tnp.health.database();
    console.log('Database Status:', dbHealth);

    // Check all health endpoints
    const allHealth = await tnp.health.getAll();
    console.log('All Health Checks:', allHealth);
    ```

  </Step>

  <Step title="Explore available resources">
    
    The SDK provides organized access to all Tonight Pass API resources:

    ```typescript
    // User management
    const currentUser = await tnp.users.me();
    console.log('Current User:', currentUser);

    // Get a specific user
    const user = await tnp.users.get('user-id');
    console.log('User:', user);

    // Check username availability
    const available = await tnp.users.check('username', true);
    console.log('Username Available:', available);
    ```

  </Step>
</Steps>

## Authentication

Tonight Pass SDK supports multiple authentication methods:

### Sign In

```typescript
const response = await tnp.auth.signIn({
  email: 'user@example.com',
  password: 'your-password'
});

if (response.success) {
  // Update client with access token
  tnp.client.setAccessToken(response.data.accessToken);
}
```

### Sign Up

```typescript
const response = await tnp.auth.signUp({
  email: 'user@example.com',
  username: 'username',
  password: 'your-password',
  firstName: 'John',
  lastName: 'Doe'
});

if (response.success) {
  console.log('Account created successfully');
}
```

### OAuth2

Connect with OAuth2 providers like Google:

```typescript
// Get OAuth2 authorization URL
const authUrl = tnp.auth.oauth2.connect('google', {
  redirectUri: 'https://yourapp.com/callback'
});

// Redirect user to authUrl
window.location.href = authUrl;

// After callback, handle Google One Tap
const response = await tnp.auth.oauth2.googleOneTap({
  credential: 'google-credential-token'
});
```

### Refresh Token

```typescript
const response = await tnp.auth.refreshToken();

if (response.success) {
  tnp.client.setAccessToken(response.data.accessToken);
}
```

## Using with React

The `@tonightpass/react` package provides React hooks powered by SWR for data fetching:

<Steps>
  <Step title="Install React SDK">
    
    ```bash
    npm install @tonightpass/react
    ```

  </Step>

  <Step title="Use the useAPI hook">
    
    The `useAPI` hook automatically handles caching, revalidation, and error handling:

    ```typescript
    import { useAPI } from '@tonightpass/react';

    function UserProfile({ userId }: { userId: string }) {
      const { data, error, isLoading } = useAPI('/users/@:userId', { userId });

      if (isLoading) return <div>Loading...</div>;
      if (error) return <div>Error: {error.message}</div>;

      return (
        <div>
          <h1>{data.username}</h1>
          <p>{data.email}</p>
        </div>
      );
    }
    ```

  </Step>

  <Step title="Configure the client">
    
    Set the access token for authenticated requests:

    ```typescript
    import { client } from '@tonightpass/react';

    // After authentication
    client.setAccessToken('your-access-token');
    ```

  </Step>
</Steps>

## Working with Organizations

Manage organizations and events:

```typescript
// Get all organizations
const organizations = await tnp.organizations.getAll();

// Get a specific organization
const org = await tnp.organizations.get('org-id');

// Get organization events
const events = await tnp.organizations.events.getAll('org-id');

// Get a specific event
const event = await tnp.organizations.events.get('org-id', 'event-id');
```

## Handling Responses

All SDK methods return responses in a consistent format:

```typescript
type SuccessfulAPIResponse<T> = {
  success: true;
  data: T;
};

type ErroredAPIResponse = {
  success: false;
  message: string;
  statusCode: number;
  errors?: {
    [key: string]: string[];
  };
};
```

Example with error handling:

```typescript
try {
  const user = await tnp.users.me();
  console.log('User:', user);
} catch (error) {
  if (error instanceof TonightPassAPIError) {
    console.error('API Error:', error.message);
    console.error('Status:', error.status);
    console.error('Errors:', error.data.errors);
  }
}
```

## Caching

Enable response caching to improve performance:

```typescript
const tnp = new TonightPass({
  baseURL: 'https://api.tonightpass.com',
  cache: {
    enabled: true,
    ttl: 60000, // Cache for 60 seconds
    maxSize: 100 // Maximum cache entries
  }
});

// Clear cache manually
tnp.client.clearCache();

// Get cache statistics
const stats = tnp.client.getCacheStats();
console.log('Cache stats:', stats);
```

## Full Example

Here's a complete example putting it all together:

```typescript
import { TonightPass } from 'tonightpass';

async function main() {
  // Initialize client
  const tnp = new TonightPass({
    baseURL: 'https://api.tonightpass.com',
    cache: {
      enabled: true,
      ttl: 60000
    }
  });

  try {
    // Check API health
    const health = await tnp.health.api();
    console.log('API Status:', health);

    // Sign in
    const signInResponse = await tnp.auth.signIn({
      email: 'user@example.com',
      password: 'password'
    });

    if (signInResponse.success) {
      // Set access token
      tnp.client.setAccessToken(signInResponse.data.accessToken);

      // Get current user
      const user = await tnp.users.me();
      console.log('Welcome,', user.username);

      // Get user's organizations
      const orgs = await tnp.organizations.getAll();
      console.log('Your organizations:', orgs);
    }
  } catch (error) {
    console.error('Error:', error);
  }
}

main();
```

<Note>
Remember to handle errors appropriately in production applications and never commit sensitive credentials to version control.
</Note>
