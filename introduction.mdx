## What is Tonight Pass?

Tonight Pass is a platform reshaping how people discover and experience events. Whether you're organizing a concert, managing a club night, or looking for tonight's best events, Tonight Pass provides the infrastructure to make it happen.

The platform features real-time ticketing, seamless event management, and powerful organization tools, all designed to bring people together through unforgettable experiences. For organizers, we provide everything needed to create, manage, and grow their events. For attendees, we offer a modern way to discover and access the best events in their city.

<Warning>
These packages are currently for internal Tonight Pass applications only and are not yet ready for external use. Future versions will include an API key system for public access.
</Warning>

## Key Features

- **Type-Safe Client** - Full TypeScript support with comprehensive type definitions
- **Multi-Framework Support** - SDKs for Node.js, React, and NestJS
- **Authentication** - Built-in OAuth2 support with multiple providers (Google, etc.)
- **User Management** - Complete user profile and account management
- **Event Management** - Create, update, and manage events and organizations
- **Real-Time Features** - WebSocket support for channels and notifications
- **Caching** - Optional response caching for improved performance
- **Feed & Social** - User posts, comments, reposts, and social feed features
- **Bookings & Orders** - Complete ticketing and order management
- **Health Checks** - Built-in health check endpoints for monitoring

## Available Packages

### Core SDK

**`tonightpass`** - The main Node.js SDK package providing the core API client and type definitions.

```bash
npm install tonightpass
```

### Framework-Specific SDKs

**`@tonightpass/react`** - React hooks and components for easy integration in React applications.

```bash
npm install @tonightpass/react
```

**`@tonightpass/nest`** - NestJS decorators and modules for server-side integration.

```bash
npm install @tonightpass/nest
```

**`@tonightpass/nestjs-mailjet`** - NestJS module for integrating Mailjet email service.

```bash
npm install @tonightpass/nestjs-mailjet
```
