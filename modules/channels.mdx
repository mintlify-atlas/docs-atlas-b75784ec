The Channels module provides methods to create and manage communication channels for users and organizations, including participant management and messaging functionality.

## Channel Management

### me

Get all channels for the authenticated user.

```typescript
const myChannels = await tonightpass.channels.me({
  limit: 20,
  offset: 0,
});
```

<ParamField path="options" type="ArrayOptions<Channel>">
  Pagination and filtering options
</ParamField>

**Returns:** `Promise<Channel[]>`

---

### getByOrganization

Get all channels for a specific organization.

```typescript
const orgChannels = await tonightpass.channels.getByOrganization(
  'my-organization',
  { limit: 20 }
```

## Messages

The channels module includes a `messages` submodule for managing channel messages, including creating, updating, deleting messages, and managing reactions.

```typescript
// Access channel messages
const messages = await tonightpass.channels.messages.getAll(channelId);
```

## Usage Example

<CodeGroup>
```typescript Complete Channel Workflow
import { TonightPass } from 'tonightpass';

const tonightpass = new TonightPass({
  apiKey: process.env.TONIGHTPASS_API_KEY,
});

// Create a new channel
const channel = await tonightpass.channels.create({
  name: 'Project Alpha',
  description: 'Discussions about Project Alpha',
  type: 'private',
});

// Add participants
await tonightpass.channels.addParticipant(channel.id, {
  username: 'alice',
});

await tonightpass.channels.addParticipant(channel.id, {
  username: 'bob',
});

// Get channel members
const members = await tonightpass.channels.getMembers(channel.id);
console.log(`Channel has ${members.length} members`);

// Send a message
await tonightpass.channels.messages.create(channel.id, {
  content: 'Welcome to the channel!',
});

// Get all messages
const messages = await tonightpass.channels.messages.getAll(channel.id);
```
</CodeGroup>

<Note>
  Channels support both user-owned and organization-owned contexts. Use the appropriate methods based on your use case.
</Note>
