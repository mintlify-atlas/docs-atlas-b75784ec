## Prerequisites

Before installing Tonight Pass SDK, ensure you have the following:

- **Node.js** 14.0.0 or higher
- **Package Manager** - npm, yarn, or pnpm

## Core SDK Package

The main `tonightpass` package provides the core API client and type definitions for Node.js environments.

<CodeGroup>

```bash npm
npm install tonightpass
```

```bash yarn
yarn add tonightpass
```

```bash pnpm
pnpm add tonightpass
```

</CodeGroup>

### Dependencies

The core package includes the following dependencies:

- `@mikro-orm/core` - Database ORM integration
- `@nestjs/terminus` - Health check endpoints
- `class-transformer` & `class-validator` - Data validation
- `pathcat` - URL path construction
- `redaxios` - HTTP client
- `reflect-metadata` - Metadata reflection
- `stripe` - Payment processing

## React SDK

For React applications, install the `@tonightpass/react` package which provides React hooks powered by SWR.

<CodeGroup>

```bash npm
npm install @tonightpass/react
```

```bash yarn
yarn add @tonightpass/react
```

```bash pnpm
pnpm add @tonightpass/react
```

</CodeGroup>

### Peer Dependencies

<Note>
The React SDK requires React as a peer dependency.
</Note>

```json
{
  "peerDependencies": {
    "react": "*"
  }
}
```

Make sure you have React installed in your project:

<CodeGroup>

```bash npm
npm install react
```

```bash yarn
yarn add react
```

```bash pnpm
pnpm add react
```

</CodeGroup>

## NestJS SDK

For NestJS applications, install the `@tonightpass/nest` package for server-side integration.

<CodeGroup>

```bash npm
npm install @tonightpass/nest
```

```bash yarn
yarn add @tonightpass/nest
```

```bash pnpm
pnpm add @tonightpass/nest
```

</CodeGroup>

### Peer Dependencies

<Note>
The NestJS SDK requires @nestjs/common version 8.x or higher.
</Note>

```json
{
  "peerDependencies": {
    "@nestjs/common": ">=8.x.x"
  }
}
```

If you don't have NestJS installed:

<CodeGroup>

```bash npm
npm install @nestjs/common @nestjs/core
```

```bash yarn
yarn add @nestjs/common @nestjs/core
```

```bash pnpm
pnpm add @nestjs/common @nestjs/core
```

</CodeGroup>

## NestJS Mailjet Module

For email functionality with Mailjet in NestJS applications, install the `@tonightpass/nestjs-mailjet` package.

<CodeGroup>

```bash npm
npm install @tonightpass/nestjs-mailjet
```

```bash yarn
yarn add @tonightpass/nestjs-mailjet
```

```bash pnpm
pnpm add @tonightpass/nestjs-mailjet
```

</CodeGroup>

### Peer Dependencies

<Warning>
The NestJS Mailjet module requires several peer dependencies.
</Warning>

```json
{
  "peerDependencies": {
    "@nestjs/axios": ">=1.x.x",
    "@nestjs/common": ">=8.x.x",
    "@nestjs/core": ">=8.x.x",
    "@nestjs/platform-fastify": ">=9.x.x",
    "node-mailjet": ">=6.x.x"
  }
}
```

Install all required peer dependencies:

<CodeGroup>

```bash npm
npm install @nestjs/axios @nestjs/common @nestjs/core @nestjs/platform-fastify node-mailjet
```

```bash yarn
yarn add @nestjs/axios @nestjs/common @nestjs/core @nestjs/platform-fastify node-mailjet
```

```bash pnpm
pnpm add @nestjs/axios @nestjs/common @nestjs/core @nestjs/platform-fastify node-mailjet
```

</CodeGroup>

## TypeScript Configuration

Tonight Pass SDK is built with TypeScript and includes full type definitions. For the best development experience, ensure your `tsconfig.json` includes:

```json tsconfig.json
{
  "compilerOptions": {
    "target": "ES2020",
    "module": "commonjs",
    "lib": ["ES2020"],
    "declaration": true,
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true,
    "forceConsistentCasingInFileNames": true,
    "experimentalDecorators": true,
    "emitDecoratorMetadata": true
  }
}
```

## Verification

Verify your installation by importing the SDK in your project:

```typescript
import { TonightPass } from 'tonightpass';

const tnp = new TonightPass({
  baseURL: 'https://api.tonightpass.com'
});

console.log('Tonight Pass SDK installed successfully!');
```

For React:

```typescript
import { useAPI } from '@tonightpass/react';

console.log('Tonight Pass React SDK installed successfully!');
```
